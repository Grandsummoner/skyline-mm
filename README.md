# maybe

A VCV Rack plugin that also runs on the 4MS MetaModule.

## Modules

### Skyline
A utility module with audio, CV and gate I/O.
- Audio input scaled by level knob
- CV and gate pass through
- Compatible with 4MS MetaModule
